# 🚀 Getting Started

Welcome! This section helps you prepare your environment and mindset to begin working through the labs in this repo.

## What you need

### Basic skills recommended
- Command-line usage (bash, PowerShell, terminal)
- Basic Python (reading/editing scripts)
- Curiosity and willingness to break/test things

### Setup your lab safely
> Always work in a **controlled environment**, such as:
- Virtual Machine (e.g. VirtualBox, VMware)
- Isolated Linux environment (Kali, Parrot OS)
- Sandboxed Windows VM
- Your main OS for the most crazy of you

## Labs structure

Each folder in this repository contains:
- `README.md` → instructions
- Code files (`.py`, `.sh`, etc.)
- Challenges or questions
- Outputs or logs

You’re expected to:
- Read + run code
- Modify or break things
- Understand how attackers think
- Learn how defenders respond
