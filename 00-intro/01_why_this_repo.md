# 💡 Why this repo?

Cybersecurity is hard to learn without practice.  
There are too many resources that are:
- too theoretical
- too shallow
- or too shady

This repo was created to:
- Teach real cybersec fundamentals
- Focus on detection, analysis, and behavior
- Give blue teamers offensive knowledge (for defense)

If you're just starting, you’re in the right place. If you're already in cyber, you’ll still learn a few tricks.
