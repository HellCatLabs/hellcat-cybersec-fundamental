# 🌐 02 - Network Operations

In this module, you'll learn basic offensive networking techniques often used by attackers and malware — but for educational and detection purposes.

You'll cover:
- Interface and IP enumeration
- Port scanning
- Packet sniffing
- ARP spoofing
- Man-in-the-Middle simulation

Each folder contains:
- A script
- A README with explanation
- (Sometimes) Blue team notes on how to detect the behavior

> ⚠️ Many of these labs require **root privileges** and should be run in an isolated lab environment or virtual machine.

You are responsible for how you use this knowledge.
